# AI Content Master (SGE Enabled)

Transform your WordPress content creation with advanced AI-powered tools optimized for the modern search landscape, including Google's **Search Generative Experience (SGE)**.

## 🚀 Features

- **SGE Article Generator**: Creates full-length, high-quality articles designed to win in "AI Overviews." Includes direct answer blocks, EEAT signals, and structured data-friendly layouts.
- **AI Search Analysis**: A deep-dive SEO analyzer that assesses your content for SGE potential, Featured Snippet readiness, and E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness).
- **High-CTR Meta Generator**: Crafts compelling meta descriptions specifically designed to encourage clicks from AI summaries and traditional search results.
- **Content Refinement**:
  - **Rephrase Selection**: Highlight any text in the editor to instantly rewrite it with better flow.
  - **Article Rewriter**: Completely overhaul an existing post while maintaining its core message.
- **OpenRouter Integration**: Access the world's best AI models (OpenAI, Claude, Llama 3, Gemini) through a single API key.

## 🛠️ Installation

1. Upload the `ai-content-master` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin via the 'Plugins' menu.
3. Go to **Settings > AI Content Master** to enter your OpenRouter API Key.

## 🤖 AI Search (SGE) Strategy

This plugin implements the latest findings in AI search optimization:
- **Direct Answers**: Positions clear answers at the start of articles to increase citation probability in AI summaries.
- **Semantic Connectivity**: Ensures your content covers adjacent entities and "People Also Ask" triggers.
- **EEAT Proofing**: Encourages the use of first-hand experience and unique value propositions that pure AI-generated content often lacks.
